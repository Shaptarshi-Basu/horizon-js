Horizon-js

JavaScript port of Openstack Horizon.

Download this project as a .zip file Download this project as a tar.gz file

Introduction

Horizon-js is a Backbone-based web application that provides same functionality than OpenStack Horizon component. It aims to improve user experience in the near future by dynamically loading different parts of the web.

It has been initially developed by UPM (Universidad Politecnica de Madrid) in Spain.

Libraries

Horizon-js uses the next JS libraries: Backbone, jQuery, Underscore, Require, Bootstrap and JStack. JStack is an Openstack API implementation fully developed in JavaScript.
