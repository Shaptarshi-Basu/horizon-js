Horizon-js

JavaScript port of Openstack Horizon.

Download this project as a .zip file Download this project as a tar.gz file

Introduction

Horizon-js is a Backbone-based web application that provides same functionality than OpenStack Horizon component. It aims to improve user experience in the near future by dynamically loading different parts of the web.

It has been initially developed by UPM (Universidad Politecnica de Madrid) in Spain.

Architecture

Horizon-js uses the next JS libraries:

  • jstack: JavaScript implementation of OpenStack API
    It supports
    • Keystone User API
    • Nova API
    • Glance API
  • Backbone.js: Gives structure to web applications
    • Models with key-value binding and custom events
    • Collections with a rich API of enumerable functions
    • Views with declarative event handling
    • Connects it all to your existing API over a RESTful JSON interface
  • jQuery: Simplifies
    • HTML document traversing
    • Event handling
    • Animating
    • Ajax interactions for rapid web development.
  • Bootstrap: Responsive Design.
  • Underscore: Templates.
